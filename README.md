# Ryan Hoang - Personal Portfolio Website

## Project Description
This is a multi-page personal portfolio website designed to showcase my background as an Undergraduate Semiconductor Student and my passion for PC Hardware Assembly. The website serves as a digital CV, highlighting my technical skills, academic projects, and work experience.

## Features
* **Multi-Page Navigation:** Includes Home, About, Projects, and Contact pages with active state highlighting.
* **Responsive Design:** Fully adaptive layout that works on desktop, tablets, and mobile devices (using Flexbox and CSS Grid).
* **Project Showcase:** Detailed cards for projects like "PC Assembly Optimization" and "Semiconductor Fabrication Sim".
* **Clean UI/UX:** Uses a consistent color palette (#F2B3EE & #0877d1) and typography.

## Technologies Used
* **HTML5:** For semantic structure.
* **CSS3:** For styling, layout (Grid/Flexbox), and responsiveness.
* **VS Code:** Code editor.

## How to Run
1. Download or clone the project folder.
2. Open the folder in your file explorer.
3. Double-click on index.html to open the website in your default web browser.
4. Alternatively, you can use the "Live Server" extension in VS Code to launch the project.

## Compatibility
* **Desktop:** Chrome, Edge, Firefox, Safari.
* **Mobile:** Tested on iPhone and Android viewports (responsive breakdown at 768px).
